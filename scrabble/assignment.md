# Assignment

Create an HTML document and a CSS stylesheet to display the following Scrabble board:

![](scrabble.png)

The HTML table representing the board should be created using [Emmet abbreviations](https://docs.emmet.io/abbreviations/syntax/).

Create an alternate version of the HTML document in which the rows and columns are automatically numbered using CSS counters. The associated CSS stylesheet must reuse the original stylesheet using an `@import` at-rule.
